📄 Privacy Policy – Localhost Public URL
Effective Date: [21/02/2026]

1. Overview
Localhost Public URL is a Chrome extension designed to generate temporary public URLs for local development servers running on the user’s machine.

This extension is developed and maintained by an individual developer.

2. Data Collection
Localhost Public URL does not collect, store, transmit, or share any personal data.

Specifically:

No analytics are used

No tracking is performed

No browsing history is collected

No personal information is stored

No data is transmitted to any developer-controlled servers

3. Local Communication
The extension communicates only with:

http://127.0.0.1:54321
This is a local service running on the user’s own machine.

No data is sent to external servers by the extension.

4. Public URL Generation
Public URLs are generated using Cloudflare Tunnel technology.

The extension does not operate or control Cloudflare services.

Users are responsible for reviewing Cloudflare’s own privacy policy if desired.

5. Companion Application
The extension requires a locally installed companion application.

This application runs only on the user’s device and does not transmit data to the developer.

6. Data Storage
The extension may store a temporary authentication token locally using browser storage to communicate securely with the local service.

This data:

Is stored only on the user's device

Is not transmitted externally

Is not accessible by the developer

7. Changes to This Policy
This policy may be updated in future versions of the extension. Updates will be reflected with a revised effective date.
